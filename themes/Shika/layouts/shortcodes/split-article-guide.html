How to Use Split Article with Connector

Use the split-article shortcode to create a side-by-side comparison with a connector note:

{{< split-article 
    left="## Left Section Content
    
This is the content for the left section.
    
* Point 1
* Point 2"
    
    right="## Right Section Content
    
This is the content for the right section.
    
1. Step one
2. Step two"
    
    leftPoint="Left section key point"
    rightPoint="Right section key point"
    note="**Important connection:** This note highlights the relationship between both sections."
    showConnector="true"
    direction="horizontal" >}}

Parameters:

  • left - Content for the left section (supports Markdown)
  • right - Content for the right section (supports Markdown)
  • leftPoint - Brief label for the left connection point
  • rightPoint - Brief label for the right connection point
  • note - The connector note text (supports Markdown)
  • showConnector - Set to "true" or "false" to show/hide the connector
  • direction - Either "horizontal" (default) or "vertical"

Vertical Layout Example:

{{< split-article
    left="Top section content"
    right="Bottom section content"
    direction="vertical"
    note="This connects the top and bottom sections" >}}
